Core Capabilities
Every Layer Secured
Built with security at every layer — from database queries to session cookies to encryption at rest.
🔑

OTP Multi-Factor Authentication

Every login and fund transfer requires a time-limited one-time password generated via cryptographically secure PRNG and encrypted with AES-256-GCM before storage.

secrets.token_urlsafe
👥

Role-Based Access Control

Three-tier RBAC (Admin / Banker / User) enforced via Python decorators on every route. Unauthorized access attempts are logged instantly to the audit trail.

@role_required
🔒

Account Lockout & Brute-Force Defense

Accounts lock for 24 hours after 3 failed login attempts. OTP also locks after 3 wrong entries. All events logged with severity levels.

MAX_FAILED_ATTEMPTS = 3
⚡

Atomic Fund Transfers

Transfers execute inside a single BEGIN EXCLUSIVE SQLite transaction — balance read, validation, debit, credit, and record all happen atomically. No TOCTOU race possible.

BEGIN EXCLUSIVE
🛡️

CSRF & Session Security

Flask-WTF CSRF tokens on all forms. SameSite=Lax cookies. 15-minute idle session timeout. Cache-Control: no-store headers prevent back-button access after logout.

flask-wtf · SameSite=Lax
📋

Security Audit Trail

Every login attempt, OTP event, fund transfer, admin action, and session timeout is logged with timestamp, IP address, username, and severity level.

log_audit() · IP · Severity
